________ink font browser________

What is ink font browser?
-this tiny extension allows you to easily browse through installed fonts on your windows OS.
-you can type in your sample text to preview.
-if you want to preview on any GUI element, you can apply the file "preview" in the Font folder and as you change the font you will be able to view the new font on your GUI element.

Please note that the Folder "Font" is only created after first startup of the extension.
